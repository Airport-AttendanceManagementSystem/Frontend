📚 Attendance Management System (Frontend)

🚀 Overview

The Attendance Management System is a full-stack web application designed to efficiently manage employee attendance, user roles, and reporting.

It is built with a modern architecture using:

🎨 Frontend: React + Vite + Chakra UI

⚙️ Backend: NestJS + TypeScript

🗄️ Database: MySQL

The system supports role-based access control (Admin/User) with secure authentication and scalable API design.


⚙️ Installation & Setup

🎨 Frontend Setup

      cd frontend
      npm install
      npm run dev
